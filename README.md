# Skjutbana

Välkommen till Skjutbana!

## Om spelet
Skjutbana är ett kostnadsfritt program för att skapa en simulering av en inomhusskjutbana. Med detta program kan du träna och förbättra dina skjutkunskaper i en kontrollerad miljö. Det är perfekt för både nybörjare och erfarna skyttar.

## Funktioner
- Användarvänligt gränssnitt
- Anpassningsbara inställningar för olika vapentyper
- Realtidsfeedback på dina skott
- Statistik och resultatrapportering
- Flerspelarläge för att tävla med vänner

## Installation
1. Klona detta repository:
   ```bash
   git clone https://github.com/jwestholm/skjutbana.git
   ```
2. Gå till mappen:
   ```bash
   cd skjutbana
   ```
3. Installera nödvändiga beroenden:
   ```bash
   npm install
   ```
4. Starta programmet:
   ```bash
   npm start
   ```

## Användning
- För att börja träna, välj en vapentyp och justera inställningarna som du önskar.
- Följ instruktionerna på skärmen för att få bästa möjliga träning.
